# API Gateway - Список изменений

## Исправления для интеграции с фронтендом

### 1. Добавлены эндпоинты аутентификации
- `POST /api/auth/token/` - получение JWT токена
- `POST /api/auth/token/refresh/` - обновление токена

### 2. Улучшена обработка файлов
- Добавлена поддержка `multipart/form-data` для загрузки файлов
- Правильная передача файлов в Media Service

### 3. Улучшена обработка ошибок
- Добавлена обработка `RequestException` для недоступных сервисов
- Улучшена обработка JSON ответов

### 4. Улучшены CORS настройки
- Добавлены разрешенные методы и заголовки
- Настроена поддержка credentials

### 5. Исправлено построение URL
- Правильное проксирование запросов к микросервисам
- Корректная передача заголовков и параметров
